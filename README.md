# 💸 Amigo do Bolso

Aplicação de controle financeiro pessoal com foco em **uso prático no dia a dia**.

Projeto originalmente criado como TCC (2021) e reescrito para resolver necessidades reais de organização financeira pessoal e familiar.

---

## 🔗 Acesse o projeto

https://amigodobolso.jmadev.com.br/

---

## 🎯 Objetivo

Facilitar o controle financeiro com:

- visão clara dos gastos  
- organização por categorias e grupos  
- acompanhamento mensal  
- geração de insights úteis (com IA)  

---

## 🚀 Funcionalidades

- Dashboard com visão geral das finanças  
- Relatórios mensais  
- Controle de cartão de crédito com parcelas  
- Despesas recorrentes  
- Notificações de contas (vencidas e a vencer)  
- Envio automático de relatório mensal  
- Controle de VR/VA  
- Grupos familiares (compartilhamento por código)  
- Categorias personalizadas  
- Metas financeiras  

---

## 🤖 Inteligência Artificial

Geração de análises com base nos dados financeiros do usuário, incluindo:

- nível de comprometimento da renda  
- identificação de gastos desorganizados  
- sugestões de economia  
- direcionamento para o próximo mês  

---

## ⚙️ Tecnologias

- PHP 8.2  
- MySQL  
- JavaScript (Vanilla)
